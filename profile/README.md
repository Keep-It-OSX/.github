## Overview

<a href="#">Keep It</a> provides unified organization for the documents, notes, and reference materials that accumulate during research and knowledge work. Rather than scattering information across multiple applications and storage systems, Keep It consolidates everything into a single, searchable library with flexible organization options.

The <a href="#">unified storage</a> accepts diverse content types. Store documents, images, web archives, and notes together. Each item regardless of type lives in the same organizational structure. Unified storage simplifies knowing where things are.

<a href="#">Folder organization</a> provides hierarchical structure. Create folder trees matching mental organization. Nest folders as deeply as needed. Folder structure scales from simple to complex libraries.

The <a href="#">tagging system</a> adds cross-cutting categorization. Apply multiple tags to any item. Find items by tag regardless of folder location. Tags reveal connections that hierarchy obscures.

<a href="#">Smart folders</a> collect items matching criteria automatically. Define search criteria and smart folders populate dynamically. New items appear automatically when matching. Smart folders provide saved searches with folder convenience.

The <a href="#">note editing</a> handles text content natively. Rich text editing for formatted notes. Markdown support for structured writing. Notes integrate with other stored content.

<a href="#">PDF handling</a> manages document collections. View PDFs within Keep It. Add annotations without external applications. PDF integration addresses common document needs.

The <a href="#">web archiving</a> preserves online content permanently. Save web pages as self-contained archives. Content remains accessible regardless of website changes. Web archiving captures ephemeral online information.

<a href="#">Search functionality</a> finds content instantly. Full-text search across all content types. Filter results by type, tag, or folder. Search makes large libraries navigable.
